jQuery Single Option Filter

Usage

  1. Use data-filter='filtername' for the trigger (filter), e.g. <a href="#" data-filter="test"> *works only with a-tags*
  2. Use data-filter-content='filtername' for the content, e.g. <div data-filter-content="test">
  3. For reset: use 'all' as value of the data-filter
